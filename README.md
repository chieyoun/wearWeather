# 📊프로젝트 개요

---

## 목표

⭐ 갑작스러운 기상 변화로 인한 불편함을 최소화.<br>
⭐ 사용자들이 각자의 지역에서 예상되는 기상 상황에 맞는 옷차림을 선택할 수 있도록 도움.<br>
⭐ 사용자의 위치 정보에 따른 온도 / 날씨 정보 제공.<br>
